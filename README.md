# 🔁 Palindrome Checker

A simple web app that checks whether a given word or phrase is a palindrome. It ignores punctuation, spaces, and letter casing to give accurate results.

## 🚀 Live Demo

> _Coming soon_ — you can run this project locally or deploy it on GitHub Pages or Netlify.

---

## 📦 Project Structure

``

palindrome-checker/
│
├── index.html       # Main HTML file
├── styles.css       # Styles for layout and elements
└── script.js        # JavaScript logic for checking palindrome

``

---

## 🧠 Features

- Checks if a word or sentence is a palindrome
- Ignores punctuation, spaces, and letter case
- Alerts if input is empty
- Clean UI with responsive design
- Handles test cases such as:
  - `eye`
  - `race car`
  - `A man, a plan, a canal. Panama`
  - `not a palindrome`

---

## 🔨 How to Use

### Option 1: Open Locally

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Type a word or phrase into the input field.
4. Click **Check** to see the result.

### Option 2: Deploy Online

You can deploy this on:
- GitHub Pages
- Netlify
- Vercel

Just drag and drop the files into the deployment platform.

## 📋 Example Inputs and Results

| Input                          | Output                                |
|-------------------------------|----------------------------------------|
| `race car`                    | `race car is a palindrome.`            |
| `A man, a plan, a canal. Panama` | `A man, a plan, a canal. Panama is a palindrome.` |
| `not a palindrome`            | `not a palindrome is not a palindrome.` |
